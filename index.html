It is more easier to handle a website layout by using CSS Grid System than CSS Flexbox System. Where CSS Flexbox is uni-directional (either column-wise or row-wise), CSS Grid System is Two-dimensional (both column-wise and row-wise) simultaneously. CSS Flexbox System is designed for layout in one dimension - either a row or a column. CSS Grid System is designed for two-dimensional layout - rows, and columns at the same time.

We use Media Query for making a website responsive. As example, if a website is developed using only HTML & CSS, normally the website will be broken while visiting the site from a smaller device like Mobile or Tablet PC. But if we use Media Query along with these, it allows us to create different layouts depending on the size of the viewport of the device; they can also be used to detect other things about the environment in which our site is running on. For example, whether the user is using a touchscreen rather than a mouse. In a sentence, Media queries are a key part of responsive web design.

CSS Box Model mainly indicates a content area where different kinds of HTML Elements can be displayed. This is optionally surrounded by padding, a border, and a margin, on one or more sides.

In CSS, the term "box model" is used when talking about design and layout. Every HTML Element that used in the website has a 'Box Model'. When we try to change an elements margin or padding or sizings, we can take help of CSS Box Model from our browsers debugging/developer tool by using 'Inspect'.

HTML Semantic Tags tags indicates the meaning of the content they contain.

For example, if we use HTML Heading One (h1) Tag or, HTML Paragraph (p) Tag or, HTML Division (div) Tag, tags like these cannot express the meaning of content they contain; but if we use an article (article) tag, here we can clearly understand and say that inside this tag some article-type paragraphs will be placed.
